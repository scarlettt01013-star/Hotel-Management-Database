# Hotel Management Database

## Project Description
This project is a simple Hotel Management Database created using MySQL. It demonstrates the basic concepts of relational databases, including table creation, data insertion, queries, and updates.

## Database Tables
- Guests
- RoomTypes
- Rooms
- Reservations

## Features
- Store guest information
- Manage room types
- Manage hotel rooms
- Create reservations
- View available rooms
- Count total reservations
- Update room status

## Technologies
- MySQL
- SQL
